



querySelector和getElementById方法的区别


querySelector:
	a 仅兼容IE8及以上
	b querySelector()是用于接收一个CSS选择符，返回与该模式匹配的第一个元素；
	c query选择符选出来的元素及元素数组是静态的.
	d 不能查找伪类元素。

querySelectorAll:
	a 仅兼容IE8及以上
	b querySelectorAll()用于选择匹配到的所有元素。
	c 同上
	d 不能查找伪类元素。

getElementById:
	a 兼容所有浏览器
	b 通过ID获得所选择的元素
	c getElement这种方法选出的元素是动态的。 会重新寻找
	d 不能查找伪类元素。




Demo :

<body>
	<ul> <li>111</li> <li>222</li> <li>333</li> </ul>
	<script type="text/javascript">
		var ul=document.querySelector('ul');
        var listQuery=ul.querySelectorAll('li');
        var ul=document.getElementsByTagName('ul')[0];
        var listDocumentBy=ul.getElementsByTagName('li');
        
        
        for(var i=0;i<listQuery.length;i++){
            ul.appendChild(document.createElement('li'));
        }//这个时候就创建了3个新的li，添加在ul列表上。
        
        
        for(var i=0;i<5;i++){
            ul.appendChild(document.createElement('li'));
        }   
        
        console.log(listQuery.length) //输出的结果仍然是3，不是此时li的数量3 + 3 + 5
        console.log(listDocumentBy.length)//此时输出的结果就是3+3+5=11
	</script>
</body>
